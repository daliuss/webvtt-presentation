What is WebVTT? Seriously, what kind of name is that?
  • WebVTT: Web Video Text Tracks
  • Format Intended for marking up Text-Track Resources
  • Open standard for timed captions
Why do we need WebVTT? Another timed caption standard
  • Furthers media accessibility on the web
  • Captions and subtitle media for impaired users
  • Simplicity in authoring a WebVTT file
  • Video content navigation using chapter markers
  • International language support using UTF-8
WebVTT in the future?
  • Provide captions for live streams
  • Provide an easy way for screen readers to access data
  • And of course karaoke

Javascript and the DOM Extending the DOM to implement WebVTT
Javascript and the DOM Extending the DOM to implement WebVTT

DOM Objects

  • <track> allows authors to specify external timed text tracks for media elements
  • TextTrack represents a single text track in the DOM
  • TextTrackList contains a list of TextTracks attached to a media element
  • TextTrackCue represents a single timed cue within a TextTrack
  • TextTrackCueList contains a list of TextTrackCues within a TextTrack
Javascript and the DOM

Sample Track

                    <video src="media/demo.webm" controls preload="auto">
                        <track kind="subtitles"
                               src="captions/demo.vtt" 
                               label="ptsubs" 
                               srclang="pt-BR" 
                               id="demotrack" default />
                    </video>
Javascript and the DOM

Getting a track element

                    var trackElement = document.getElementById("demotrack");

Creating a TextTrackCue

                    var ttc = new TextTrackCue(1, 3, "What is it like to be Brazilian?");

Adding/Removing a TextTrackCue

                    var textTrack = trackElement.track;
                    textTrack.addCue(ttc);
                    textTrack.removeCue(ttc);
